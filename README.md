# Harness Study - AI 에이전트 프레임워크 학습

> AI 에이전트 구축에 필요한 핵심 개념과 프레임워크를 체계적으로 학습할 수 있는 교육 사이트

**[https://harness-study.dreamitbiz.com](https://harness-study.dreamitbiz.com)**

---

## 프로젝트 소개

Harness Study는 AI 에이전트 프레임워크의 핵심 개념부터 실전 구축까지 체계적으로 학습할 수 있는 웹 기반 교육 플랫폼입니다. 에이전트 아키텍처, 메모리 시스템, 스킬 패턴, 팀 구성 등 실무에 필요한 모든 주제를 다룹니다.

## 학습 카테고리

| 카테고리 | 주요 내용 |
|---------|----------|
| **소개 (Intro)** | AI 에이전트 프레임워크 개요 |
| **사전 지식 (Prereqs)** | 학습 전 필요한 기초 지식 |
| **에이전트 (Agents)** | 에이전트 아키텍처 및 설계 |
| **메모리 (Memory)** | 에이전트 메모리 시스템 |
| **패턴 (Patterns)** | 에이전트 설계 패턴 |
| **스킬 (Skills)** | 에이전트 스킬/도구 구현 |
| **팀 (Teams)** | 멀티 에이전트 팀 구성 |
| **실습 (Practice)** | 실전 연습 문제 |
| **튜토리얼 (Tutorials)** | 단계별 가이드 |
| **구축 (Build)** | 에이전트 시스템 구축 |
| **커뮤니티** | Q&A, FAQ, 자유 게시판, 자료실 |

## 기술 스택

- **프레임워크:** React 19 + TypeScript
- **빌드 도구:** Vite 7
- **라우팅:** React Router DOM 7
- **백엔드:** Supabase (Auth, Database)
- **배포:** GitHub Pages
- **UI:** CSS Variables 기반 디자인 시스템, 다크모드, 5가지 컬러 테마

## 주요 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| `SEOHead` | SEO 메타 태그 관리 |
| `GuideSidebar3` | 가이드 사이드바 네비게이션 |
| `SearchModal` | 글로벌 검색 모달 |
| `StarRating` | 별점 평가 컴포넌트 |
| `ProfileCompleteModal` | 회원가입 후 프로필 정보 입력 모달 |
| `PaymentNudgePopup` | 라이선스 미보유 사용자 결제 안내 팝업 |

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# GitHub Pages 배포
npx gh-pages -d dist
```
